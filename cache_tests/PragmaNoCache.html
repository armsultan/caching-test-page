This page was sent with the Pragma: no-cache response header. It is used for backwards compatibility with HTTP/1.0 caches where the Cache-Control HTTP/1.1 header is not yet present.

This page will not be disk cached by Internet Explorer. No other modern browsers recognize the Pragma header for this purpose, and with the exception of those browsers that do not cache HTTPS content at all, will write this page to the disk cache.

                  add_header Pragma "no-cache";

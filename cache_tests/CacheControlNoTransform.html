Cache-Control: no-transform

Cache Type Will Cache?

Proxy Yes

This header is part of the HTTP/1.1 standard and instructs caches to not apply transformations or conversions to the resource and does not affect cachability or cache duration

The Content-Encoding, Content-Range, Content-Type headers must not be modified by a proxy. A non- transparent proxy might, for example, convert between image formats in order to save cache space or to reduce the amount of traffic on a slow link. The no-transform directive disallows this. 


                    # Webserver config
                    server {
                        # ...
                        location /CacheControlMustRevalidate.html {
                            add_header Cache-Control "no-transform";
                            add_header "Expires" @1h; # Set expires to 1hr from now
                        }
                    }

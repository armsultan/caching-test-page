This page was sent with the HTML tag <meta http-equiv="Cache-Control" content="no-cache">. No browsers will recognize the Cache-Control header when it is specified in a meta tag; it must be included as a "real" HTTP response header. "meta http-equiv" does not have a value of "Cache-Control" according to MDN Doc and HTML5 Docs

This tag is non-standard, but many sites continue to rely on it to prevent caching anyway. This page will not be disk cached by Internet Explorer. No other modern browsers recognize the Pragma header for this purpose, and with the exception of those browsers that do not cache HTTPS content at all, will write this page to the disk cache.

Proxy caches will ignore meta tags and follow any cache directives set in the HTTP headers

This page was sent with the Age response header containing the time in seconds the object has been in a proxy cache. It is usually calculated as a difference between the proxy's current date and the Date general header included in the HTTP response. 


                  ## Cache proxy config

                  # There is no native support for this header https://trac.nginx.org/nginx/ticket/146
                  # The best way would be to use the $upstream_http_ variable class 
                  # to get the absolute age of the resource by picking up its 
                  # Date header through $upsteam_http_date.
                  server {
                      # ...
                      location ~ /Age.html {
                        add_header X-Cache-Date $upstream_http_date;
                    }
                  }

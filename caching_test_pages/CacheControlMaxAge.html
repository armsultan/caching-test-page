Cache Type Will Cache?

This page was sent with the Cache-Control: max-age response header with a future date, for example, Cache-Control: max-age 3600. This header is part of the HTTP/1.1 standard and specifies the maximum amount of time a resource will be considered fresh. Contrary to the depreciated Expires directive, this directive is relative to the time of the request.


                    # Webserver config
                    server {
                        # ...
                        location /CacheControlMaxAge.html {
                          expires 1h; # Set expires to 1hr from now
                        }
                    }

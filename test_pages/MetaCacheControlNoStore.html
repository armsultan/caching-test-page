meta tag cache control (no-store)

This page is named MetaCacheControlNoStore.html, and was sent with the HTML tag <meta http-equiv="Cache-Control" content="no-store">. No browsers recognize the Cache-Control header when it is specified in a meta tag; it must be included as a "real" HTTP response header. With the exception of those browsers that do not cache HTTPS content at all, will write this page to the disk cache. 

        
                <!-- The http-equiv attribute provides an HTTP header for the information/value of the content attribute.
                The http-equiv attribute can be used to simulate an HTTP response header.-->
                <meta http-equiv="Cache-Control" content="no-store">

meta tag cache control (no-cache)

This page is named MetaPragmaNoCache.html, and was sent with the HTML tag <meta http-equiv="Pragma" content="no-cache">. This tag is non-standard, but many sites continue to rely on it to prevent caching anyway. This page will not be disk cached by Internet Explorer. No other modern browsers recognize the Pragma header for this purpose, and with the exception of those browsers that do not cache HTTPS content at all, will write this page to the disk cache. 

        
                <!-- The http-equiv attribute provides an HTTP header for the information/value of the content attribute.
                The http-equiv attribute can be used to simulate an HTTP response header.-->
                <meta http-equiv="Pragma" content="no-cache">

Pragma: no-cache

This page is named PragmaNoCache.html, and was sent with the response header Pragma: no-cache. This header is non-standard, but many sites continue to rely on it to prevent caching anyway. This page will not be disk cached by Internet Explorer. No other modern browsers recognize the Pragma header for this purpose, and with the exception of those browsers that do not cache HTTPS content at all, will write this page to the disk cache.

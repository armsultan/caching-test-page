Cache-Control: no-store
This page is named CacheControlNoStore.html, and was sent with the response header Cache-Control: no-store. This is the header specified in the HTTP/1.1 standard for preventing a page from being cached to persistent storage. This page will not be disk cached by any standards compliant browser. One notable exception that the Cache-Control header is not recognized by Internet Explorer 8 and earlier, if a server downgrades the connection to HTTP/1.0 when sending the response. This behavior is common among older Apache servers, including this one. 

        
        # Proxy cache server settings (override)
        server {
            # ...
            location /PragmaNoCache.html {
                add_header Cache-Control "no-store";
            }
        }
